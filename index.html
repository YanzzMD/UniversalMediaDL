API Reference

Backend API Docs

RESTful API running on http://localhost:5000/api

GET
/api/health

Check server status and yt-dlp version.

{"status": "online", "ytdlp_version": "2024.x"}
POST
/api/info

Fetch video/playlist metadata and available formats.

{"url": "https://youtube.com/watch?v=...", "playlist": false}
POST
/api/download/start

Start an async download job. Returns job_id.

{"url": "...", "audio_only": false, "format_id": "137"}
GET
/api/download/status/:job_id

Poll download progress (status, percentage, speed, ETA).

{"status": "downloading", "progress": 45.3, "speed": "2.3MiB/s"}
GET
/api/download/file/:job_id

Retrieve the completed file for download.

DEL
/api/download/cancel/:job_id

Cancel a pending/running job and clean up its files.

DEL
/api/history/:user_id

Delete a history entry and remove the file from server.
